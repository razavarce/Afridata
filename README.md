# AFRIDATA — Plataforma MPPRE

Sistema de inteligencia y gestión de relaciones **Venezuela–África** del Ministerio del Poder Popular para Relaciones Exteriores (Despacho para África).

## Versión publicada

**v10.5.5.0** — archivo principal: [`index.html`](index.html)

## Demo en línea

**https://razavarce.github.io/Afridata/**

Abra ese enlace en Chrome o Edge. Credenciales de prueba: `admin` / `admin`.

## Archivos del sitio

| Archivo | Uso |
|---------|-----|
| `index.html` | Página que sirve GitHub Pages |
| `html` | Copia espejo (compatibilidad con enlaces antiguos) |
| `pagina afridataV10.5.5.html` | Archivo maestro de desarrollo en local |

## Cómo publicar cambios

1. Edite `pagina afridataV10.5.5.html` en su PC.
2. Doble clic en **`PUBLICAR-GITHUB-PAGES.cmd`** (no use Git Bash para `.cmd`).
3. Espere el mensaje **LISTO**. La web se actualiza en 1–2 minutos.

Si solo necesita traer cambios de GitHub sin publicar: use `SUBIR-AHORA.cmd`.
